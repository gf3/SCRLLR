Usage

LOL NO U

// SCRLLR wants to know all about your slides/sections
// So first up, let's tell it what a slide is:
var scroll = $('section.slide').scrllr()

// There are a few ways to change slides using the `show` method.
// You can use the relative shortcuts ('first', 'prev', 'next', 'last'):
scroll.show('last')

// You can use a CSS selector:
scroll.show('#rolf-container section[rel="lol"]')

// And finally, you can simply use the numerical index (zero-based):
scroll.show(3)

// Likely, this would be in a link event handler somewhere, like so:
$('a.next').live('click', function() {
  scroll.show('next')
  return false
})

Docs

SCRLLR is meant to make fancy-ass page animations easy to manage. As you can see, it's pretty easy to use. Style your sections/slides with CSS; position them anywhere, and SCRLLR will take care of the rest.

It's is pretty smart, too. SCRLLR can determine which slide a user is on when the page loads, or even when a user scrolls the page. This allows you (the developer) to use relative controls like next and prev.

Options

There are a number of options that you can set on the SCRLLR class, instances, or even to the show method. Options and their defaults:

easing ( String )

Default: swing, Fallback: linear
Name of jQuery easing method.

duration ( Number )

Default: 400
Duration of section transition animation.

top ( Number )

Default: 0
Adjust position of the top offset. Relative to the section position. Can be a positive or negative number. E.g. -175 or 300.

left ( Number )

Default: 0
Adjust position of the left offset. Relative to the section position. Can be a positive or negative number. E.g. -175 or 300.

complete ( Function )

Default: jQuery.noop
Hollaback function called when section animation is complete.

Instance Members

SCRLLR#show( section [, options] ) → SCRLLR
  1. section (String Number) – Keyword, CSS selector, or numerical index of the section to show. Keyword being one of first, prev, next, and last.
  2. options (Object) – Configuration object associated with the current call to show.

Display the specified section by scrolling to it's position all sexy-like.

All sections currently associated with the instance.
